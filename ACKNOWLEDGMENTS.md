## File Permissions System

Our file permissions system is directly inspired by VSCodium's approach to secure file access, allowing users to:
- Explicitly trust or deny access to specific paths
- Receive prompts before allowing access to new locations
- Manage permissions through an intuitive graphical interface

## A Note on VSCodium

[VSCodium](https://github.com/VSCodium/vscodium) is an open-source project that builds Microsoft's VS Code source code into freely-licensed binaries with telemetry disabled. Their work has made it possible for developers around the world to use a powerful code editor while respecting user privacy and software freedom.

According to the VSCodium project: _"VSCodium is a community-driven, freely-licensed binary distribution of Microsoft's editor VS Code."_ The project maintains builds for Windows, Linux, and macOS, and has been instrumental in providing a more privacy-focused alternative to VS Code.
